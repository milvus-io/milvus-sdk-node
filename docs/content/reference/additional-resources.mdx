## Related Tools

### Attu - GUI for Milvus

[Attu](https://github.com/zilliztech/attu) is a modern GUI for managing Milvus:

- Visual collection management
- Data browsing and editing
- Query and search interface
- Performance monitoring

### Feder - Index Visualization

[Feder](https://github.com/zilliztech/feder) visualizes ANN indexes:

- Index structure visualization
- Performance analysis
- Index optimization tools
