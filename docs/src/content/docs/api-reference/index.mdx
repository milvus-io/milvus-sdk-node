Complete API reference for the Milvus Node.js SDK, organized by feature area. This section is intentionally separate from the documentation guide sidebar so you can browse method signatures without mixing API lookups into the learning path.

## Client

- [MilvusClient](/milvus-sdk-node/api-reference/client): constructor, SDK metadata, and client-level entry points.

## Operations

- [Collection Operations](/milvus-sdk-node/api-reference/collections): create, describe, load, release, rename, snapshot, and manage collections.
- [Data Operations](/milvus-sdk-node/api-reference/data): insert, upsert, query, search, delete, iterators, compaction, and bulk import.
- [Index Operations](/milvus-sdk-node/api-reference/indexes): create, inspect, alter, and drop vector or scalar indexes.
- [Partition Operations](/milvus-sdk-node/api-reference/partitions): create, list, and drop collection partitions.
- [Database Operations](/milvus-sdk-node/api-reference/databases): create, list, describe, alter, and drop databases.
- [User & Role Operations](/milvus-sdk-node/api-reference/users-roles): create users, update passwords, create roles, and grant privileges.
- [Resource Group Operations](/milvus-sdk-node/api-reference/resource-groups): create resource groups, inspect capacity, and transfer nodes or replicas.
- [System Operations](/milvus-sdk-node/api-reference/system): check health, inspect versions, test analyzers, and get metrics.

## Types

- [Types & Enums](/milvus-sdk-node/api-reference/types-and-enums): common data types, error codes, and response shapes.
